# Vue Calculator
## Instalação
### Com npm ou yarn:
```
npm install
or yarn install
```
## Rodar o projeto em modo de desenvolvimento

```
npm run serve
```

## Rodar os testes
```
npm t
```

## Observação
```
É necessário ter o Nodejs instalado no computador, juntamente com o npm ou yarn =)

```
